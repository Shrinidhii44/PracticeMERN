#  MERN Stack Practice-1

This repository is dedicated to mastering the **MERN (MongoDB, Express, React, Node.js)** stack. 

Currently, the project focuses on building a solid **Frontend Foundation** using semantic HTML5, modern CSS3, and JavaScript before transitioning into full-stack integration.

---

##  Project Structure
A multi-page layout designed for clean navigation and responsive design:

| File | Description |
| :--- | :--- |
| `index.html` | Home page featuring a professional Hero section. |
| `about.html` | Information about the project and its goals. |
| `services.html` | A showcase of services/features offered. |
| `contact.html` | Contact details and lead generation structure. |
| `style.css` | Global styling using Flexbox and CSS variables. |
| `script.js` | Placeholder for DOM manipulation and logic. |

---

##  Built With
* **HTML5:** Semantic structure for SEO and accessibility.
* **CSS3:** Custom layouts with a focus on responsiveness.
* **JavaScript:** Vanilla JS for future interactive components.
